## 🎯 MVP 目标（30 天核心功能）

### 核心功能定义（3-5 个）

| 功能 | 描述 | 优先级 | 工作量 |
|------|------|--------|--------|
| **1. 语音/文字双模交互** | 支持语音输入和文字输入，AI 自动识别意图并回复 | P0 | 5 人天 |
| **2. RAG 知识库问答** | 基于客户提供的文档/数据，AI 精准回答业务问题 | P0 | 8 人天 |
| **3. 数字孪生数据查询** | 自然语言查询孪生系统中的实时数据（能耗、人流、设备状态等） | P0 | 6 人天 |
| **4. 智能导览讲解** | 根据访客位置/兴趣，主动推送讲解内容 | P1 | 5 人天 |
| **5. 基础数据分析** | 简单统计查询（如今日人流、本周能耗对比） | P1 | 4 人天 |

**MVP 范围说明：**
- 聚焦核心交互能力（语音/文字 + 知识库 + 数据查询）
- 暂不包含：预测分析、多系统集成、复杂工作流
- 支持 2 个试点客户并行部署

### 技术栈选择

#### 复用现有能力

| 能力 | 来源 | 复用方式 |
|------|------|----------|
| **数字孪生平台** | 谷风现有项目 | API 对接（数据查询、场景控制） |
| **3D 可视化引擎** | 谷风自有/第三方 | 嵌入 AI 交互入口（悬浮窗/语音按钮） |
| **客户数据** | 试点客户现有系统 | 通过 API/数据库对接 |
| **部署环境** | 客户现有服务器/云 | 复用现有基础设施 |

#### 新增技术组件

| 组件 | 技术选型 | 理由 | 成本 |
|------|----------|------|------|
| **语音识别（ASR）** | 阿里云智能语音 | 准确率高（95%+）、中文优化、价格适中 | ¥0.02/分钟 |
| **大模型（LLM）** | 阿里云通义千问/Qwen | 中文能力强、支持 RAG、企业级 SLA | ¥0.008/千 tokens |
| **语音合成（TTS）** | 阿里云智能语音 | 自然度高、多音色可选、与 ASR 同平台 | ¥0.04/分钟 |
| **RAG 引擎** | 自研（LangChain+ 向量库） | 灵活定制、数据可控、成本优化 | 开发成本 |
| **向量数据库** | Milvus/阿里云 DashVector | 高性能、支持中文嵌入、易部署 | ¥500-2000/月 |
| **后端框架** | Python FastAPI | 开发效率高、异步支持、生态丰富 | 开源 |
| **前端组件** | Vue3 + 自研 UI 组件 | 与现有孪生平台风格一致 | 开发成本 |
