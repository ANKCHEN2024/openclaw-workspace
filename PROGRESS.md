## 技术栈

### 后端
- **运行时**: Node.js 20+
- **框架**: Express.js
- **数据库**: PostgreSQL + Prisma ORM
- **认证**: JWT (jsonwebtoken)
- **加密**: bcrypt
- **邮件**: Nodemailer
- **测试**: Jest + Supertest

### 前端
- **框架**: Vue 3 + Composition API
- **构建**: Vite
- **UI**: Element Plus
- **路由**: Vue Router
- **状态管理**: Pinia
- **HTTP**: Axios

## 环境配置

### 后端环境变量

```bash
# 服务器
PORT=3000
NODE_ENV=development

# 数据库
DATABASE_URL="postgresql://user:password@localhost:5432/ai_drama?schema=public"
DB_AVAILABLE=false

# JWT
JWT_SECRET="your-secret-key-change-in-production"
JWT_EXPIRES_IN="7d"

# 邮件服务
SMTP_HOST=smtp.example.com
SMTP_PORT=587
SMTP_USER=your-email@example.com
SMTP_PASS=your-password
SMTP_FROM=noreply@example.com

# 前端 URL
FRONTEND_URL=http://localhost:5173
```

```bash
VITE_API_BASE_URL=http://localhost:3000/api
```
