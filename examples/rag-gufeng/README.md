## 🔧 技术实现

### 向量嵌入

使用 DashScope (通义千问) 的嵌入模型：
- 模型：text-embedding-v2
- 维度：768
- 支持中文优化

### 向量检索

使用 LanceDB：
- 本地部署，零成本
- 支持 ANN 近似最近邻搜索
- 混合检索（向量 + 关键词）

### LLM 生成

使用 DashScope (通义千问)：
- 模型：qwen-plus
- 上下文：8K tokens
- 支持中文优化
