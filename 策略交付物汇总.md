## 🎯 推荐策略组合（核心要点）

### 策略名称
**趋势回调 + 价格行为 + 分批止盈**

### 核心逻辑
```
H1 确定趋势 → 等待回调 → M15 价格行为信号 → 顺势入场
```

### 关键参数
| 参数 | 建议值 |
|-----|-------|
| 趋势判断 | H1: EMA(20,50,200) 多头/空头排列 |
| 回调位置 | EMA20/50 或 Fibonacci 50%/61.8% |
| 入场信号 | M15 Pin Bar 或 吞没形态 |
| 止损设置 | 结构位±$3-5 或 1.5×ATR（取较近） |
| 止盈策略 | 50%@1:1.5 + 50%@1:3，1:1 后移保本 |
| 单笔风险 | 1%（最大 2%） |
| 日损上限 | 3% |
| 最佳时段 | 15:00-23:00（欧盘 + 美盘） |

### 预期表现
| 指标 | 目标值 |
|-----|-------|
| 胜率 | 55-65% |
| 盈亏比 | 1:2.5 - 1:3+ |
| 月收益率 | 5-15% |
| 最大回撤 | <15% |

## ⚠️ 重要提醒

### 风险警示
```
❗ 外汇保证金交易存在重大风险，可能导致全部本金损失
❗ 历史表现不代表未来结果
❗ 本策略仅供参考，不构成投资建议
❗ 交易前请确保充分了解风险并具备风险承受能力
```

### 成功要素
```
✅ 严格止损（永不移动扩大）
✅ 控制仓位（单笔不超过 1-2%）
✅ 执行纪律（按策略信号，不情绪化）
✅ 持续学习（复盘、记录、改进）
✅ 耐心资本（使用闲置资金，不急用）
```

### 常见错误
```
❌ 不止损或移动止损扩大风险
❌ 亏损后加仓摊平
❌ 超出风险限额交易
❌ 重大数据发布时交易
❌ 情绪化交易（愤怒/兴奋）
❌ 过度交易（无信号强行入场）
❌ 持仓过夜（除非轻仓且有充分理由）
```
