## 🎯 核心特性

### 1. 自主决策
Agent 根据当前生存状态自主决定行动策略：
- **正常状态**: 主动执行任务、观察环境
- **低算力**: 优先完成已有任务，减少新行动
- **临界状态**: 仅观察，保存资源
- **死亡**: 停止所有活动

### 2. 成本感知
每次行动都有成本，Agent 必须权衡：
- 思考：1 单位
- 行动：5 单位
- 观察：2 单位
- API 调用：10 单位
- 文件操作：3 单位

### 3. 状态持久化
使用 SQLite 持久化所有关键数据：
- Agent 状态（余额、行动计数等）
- 成本记录（每笔花费详情）
- 观察记录（每次观察结果）

### 4. 可扩展性
- 任务类型易于扩展
- 配置完全可定制
- 支持多 Agent 实例
