## 快速入门

### 1. 注册/登录
1. 访问 http://localhost
2. 点击"注册"创建新账户
3. 填写用户名、邮箱和密码
4. 登录系统

### 2. 创建第一个项目
1. 点击"新建项目"
2. 填写项目信息：
   - 项目名称
   - 项目描述
   - 小说文本（粘贴或上传）
   - 集数设置
3. 点击"创建项目"

### 3. 生成短剧
1. 进入项目详情页
2. 点击"开始生成"
3. 等待 AI 处理完成
4. 查看和编辑生成的内容
5. 导出最终视频

## 详细教程

### 教程 1：创建并配置项目

#### 步骤 1：创建新项目
1. 登录后点击"新建项目"按钮
2. 填写项目基本信息：
   ```
   项目名称：我的第一部短剧
   项目描述：根据小说改编的短剧
   集数：10 集
   每集时长：60 秒
   视频比例：9:16（竖屏）
   视频质量：1080p
   ```
3. 粘贴小说文本或上传小说文件
4. 点击"创建项目"

#### 步骤 2：配置项目设置
1. 进入项目详情页
2. 点击"项目设置"
3. 配置 AI 提供商偏好
4. 配置视频生成参数
5. 保存设置

---

### 教程 2：剧本分析

#### 步骤 1：启动剧本分析
1. 在项目详情页点击"分析剧本"
2. 确认分析参数
3. 点击"开始分析"

#### 步骤 2：查看分析结果
1. 等待分析完成（通常需要 1-5 分钟）
2. 查看识别出的人物列表
3. 查看提取的场景列表
4. 查看分集规划

#### 步骤 3：编辑分析结果
1. 点击人物卡片编辑人物信息
2. 点击场景卡片编辑场景信息
3. 调整分集规划
4. 保存修改

---

### 教程 3：人物设计

#### 步骤 1：编辑人物信息
1. 在项目详情页点击"人物"标签
2. 点击某个人物进入编辑页
3. 填写或修改：
   - 人物姓名
   - 人物描述
   - 外貌特征
   - 性格特点
   - 年龄范围
   - 性别

#### 步骤 2：生成人物图像
1. 在人物编辑页点击"生成图像"
2. 选择图像风格：
   - 写实风格
   - 动漫风格
   - 插画风格
3. 调整生成参数
4. 点击"生成"
5. 等待生成完成
6. 选择满意的图像

---

### 教程 4：分镜生成

#### 步骤 1：自动生成分镜
1. 进入某一集的详情页
2. 点击"生成分镜"
3. 选择分镜风格
4. 点击"开始生成"

#### 步骤 2：编辑分镜
1. 查看生成的分镜列表
2. 点击某个分镜进行编辑：
   - 镜头描述
   - 视觉描述
   - 镜头类型
   - 摄像机角度
   - 摄像机运动
   - 持续时间
   - 对话内容
   - 动作描述
3. 保存修改

#### 步骤 3：调整分镜顺序
1. 在分镜列表中拖拽调整顺序
2. 或使用"上移/下移"按钮
3. 保存顺序

---

### 教程 5：视频生成和导出

#### 步骤 1：生成视频片段
1. 在项目详情页点击"生成视频"
2. 选择要生成的集数
3. 配置视频生成参数：
   - AI 提供商
   - 视频质量
   - 视频比例
4. 点击"开始生成"
5. 实时查看生成进度

#### 步骤 2：配置音频
1. 进入音频配置页
2. 为每个角色选择配音音色
3. 配置背景音乐风格
4. 预览音频效果

#### 步骤 3：合成最终视频
1. 点击"合成视频"
2. 配置合成参数：
   - 过渡效果
   - 字幕样式
   - 视频格式
3. 点击"开始合成"
4. 等待合成完成

#### 步骤 4：导出视频
1. 合成完成后预览视频
2. 点击"导出视频"
3. 选择导出格式和质量
4. 下载视频文件

## API 参考

### 认证 API

#### 注册
```http
POST /api/v1/auth/register
Content-Type: application/json

{
  "username": "example",
  "email": "example@email.com",
  "password": "password123"
}
```

#### 登录
```http
POST /api/v1/auth/login
Content-Type: application/json

{
  "email": "example@email.com",
  "password": "password123"
}
```

### 项目 API

#### 创建项目
```http
POST /api/v1/projects
Content-Type: application/json
Authorization: Bearer <token>

{
  "name": "我的短剧",
  "description": "项目描述",
  "novelText": "小说全文...",
  "episodeCount": 10,
  "episodeDuration": 60,
  "videoRatio": "9:16",
  "videoQuality": "1080p"
}
```

#### 获取项目列表
```http
GET /api/v1/projects
Authorization: Bearer <token>
```

#### 获取项目详情
```http
GET /api/v1/projects/:id
Authorization: Bearer <token>
```

### 剧本分析 API

#### 分析剧本
```http
POST /api/v1/scripts/analyze
Content-Type: application/json
Authorization: Bearer <token>

{
  "projectId": "proj_123",
  "novelText": "小说文本..."
}
```

### 视频生成 API

#### 生成视频
```http
POST /api/v1/videos/generate
Content-Type: application/json
Authorization: Bearer <token>

{
  "projectId": "proj_123",
  "episodeId": "epi_456",
  "videoQuality": "1080p",
  "aiProvider": "keling"
}
```

#### 获取视频状态
```http
GET /api/v1/videos/:id/status
Authorization: Bearer <token>
```

### WebSocket 事件

#### 连接
```javascript
const ws = new WebSocket('ws://localhost/ws?token=<jwt_token>');
```

#### 订阅项目进度
```javascript
ws.send(JSON.stringify({
  action: 'subscribe',
  channel: 'project:proj_123'
}));
```

#### 接收进度更新
```javascript
ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  if (data.type === 'progress') {
    console.log('进度:', data.progress);
  }
};
```

## 常见使用问题

### Q: 生成视频需要多长时间？
A: 取决于视频长度、质量和 AI 提供商。通常 60 秒的视频需要 5-15 分钟。

### Q: 可以中途取消生成任务吗？
A: 可以，在任务详情页点击"取消"按钮即可。

### Q: 支持哪些 AI 提供商？
A: 目前支持：
- 阿里通义千问（LLM）
- 阿里通义万相（图像）
- 快手可灵 AI（视频）
- 火山即梦 AI（视频）
- 阿里云智能语音（TTS）

### Q: 如何提高生成质量？
A: 
1. 提供更详细的人物和场景描述
2. 手动调整分镜
3. 选择更高质量的生成参数
4. 多次生成选择最佳结果

### Q: 数据会保存多久？
A: 项目数据会永久保存，生成的临时文件会保留 30 天。
