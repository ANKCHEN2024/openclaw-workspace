# 👑 超级管理员指南

## 管理员账号

### 默认账号
- **用户名**: `admin`
- **密码**: `admin123`
- **邮箱**: `admin@admin.local`

### 创建新管理员

```bash
cd ~/workspace/short-video-platform/backend

# 使用默认用户名和密码
node scripts/createAdmin.js

# 自定义用户名和密码
node scripts/createAdmin.js 用户名 密码
```

---

## 管理员权限

管理员拥有以下特殊权限：

| 权限 | 说明 |
|------|------|
| 查看所有用户 | 查看系统中所有注册用户 |
| 查看所有视频 | 查看所有用户生成的视频 |
| 查看所有任务 | 查看所有生成任务 |
| 删除任何视频 | 可以删除任何用户的视频 |
| 删除用户 | 可以删除普通用户账号 |

---

## 管理后台

### 访问地址
http://localhost:8080/admin.html

### 功能

1. **仪表盘**
   - 总用户数
   - 总视频数
   - 总任务数
   - 处理中任务数

2. **用户管理**
   - 查看所有用户列表
   - 查看用户角色（管理员/普通用户）
   - 删除普通用户

---

## API 端点

管理员专用 API（需要管理员权限）：

```
GET    /api/admin/dashboard    # 仪表盘数据
GET    /api/admin/users        # 所有用户列表
GET    /api/admin/videos       # 所有视频列表
GET    /api/admin/tasks        # 所有任务列表
DELETE /api/admin/users/:id    # 删除用户
DELETE /api/admin/videos/:id   # 删除视频
```

---

## 使用流程

### 1. 登录管理后台

1. 访问 http://localhost:8080/login.html
2. 使用管理员账号登录
3. 导航栏会显示"管理"入口

### 2. 查看统计数据

登录后访问 http://localhost:8080/admin.html

### 3. 管理用户

在用户列表中可以：
- 查看用户注册时间
- 识别管理员账号
- 删除普通用户（不能删除自己）

---

## 安全提示

1. **修改默认密码** - 首次登录后请修改默认密码
2. **保护管理员账号** - 不要将管理员凭证泄露给他人
3. **谨慎删除** - 删除操作不可恢复

---

## 故障排查

### 无法访问管理后台
- 确保已登录
- 确保账号有管理员权限

### 无法删除用户
- 不能删除自己的账号
- 检查网络连接

### 统计数据不显示
- 确保后端服务已启动
- 检查浏览器控制台错误
