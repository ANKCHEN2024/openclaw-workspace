### 第二步：注册推荐平台（每个 3-5 分钟）

#### 🚀 1. 火山引擎 - 即梦 AI（推荐首选）

**免费额度**: 200 秒  
**注册链接**: https://console.volcengine.com/

**步骤**:
1. 点击「注册」
2. 输入手机号，获取短信验证码
3. 设置密码（用密码管理器生成）
4. 完成实名认证（上传身份证）
5. 登录控制台
6. 进入「访问密钥」页面
7. 点击「创建密钥」
8. 复制 Access Key 和 Secret Key（只显示一次！）
9. 保存到密码管理器

**预计用时**: 5 分钟

---

#### 🌀 2. MiniMax - 海螺 AI

**免费额度**: 150 秒  
**注册链接**: https://api.minimax.chat/

**步骤**:
1. 点击「注册/登录」
2. 手机号 + 短信验证码
3. 设置密码（密码管理器生成）
4. 完成实名认证
5. 进入控制台
6. 找到「API Keys」
7. 创建新密钥
8. 复制保存

**预计用时**: 4 分钟

---

#### ☁️ 3. 阿里云 - 通义万相

**免费额度**: 100 秒  
**注册链接**: https://dashscope.console.aliyun.com/

**步骤**:
1. 可用淘宝/支付宝账号直接登录
2. 或手机号注册新账号
3. 完成实名认证（支付宝认证最快）
4. 进入 DashScope 控制台
5. 左侧菜单「API-KEY 管理」
6. 点击「创建新的 API-KEY」
7. 复制 API 密钥

**预计用时**: 3 分钟（如有淘宝账号）

---

### 第三步：配置到平台（1 分钟）

1. 回到短视频生成平台
2. 访问 http://localhost:8080/register.html
3. 或进入「设置」页面
4. 选择对应平台
5. 粘贴 API 密钥
6. 保存配置

## 注册技巧

### 加速注册

1. **使用已有账号**
   - 淘宝账号 → 阿里云
   - 微信 → 腾讯云
   - 百度账号 → 百度智能云
   - 快手账号 → 可灵 AI

2. **准备材料**
   - 📱 手机（接收验证码）
   - 🆔 身份证（实名认证）
   - 💳 支付宝（快速实名认证）

3. **批量注册顺序**
   ```
   1. 火山引擎（额度最多）
   2. MiniMax（性价比高）
   3. 阿里云（质量最好）
   4. 快手可灵（最便宜）
   ```

### 实名认证

所有国内平台都需要实名认证，这是国家法规要求。

**快速认证方式**:
- 支付宝认证（最快，1 分钟）
- 微信认证（2-3 分钟）
- 身份证上传 + 人脸识别（5 分钟）

## 常见问题

### Q: 注册需要付费吗？
A: 完全免费！注册和获取 API 密钥都不收费，只有使用时才扣费，且有免费额度。

### Q: 学生有优惠吗？
A: 部分平台有学生认证优惠：
- 阿里云：学生机优惠
- 腾讯云：校园计划
- 火山引擎：教育优惠

### Q: 一个手机号能注册多个平台吗？
A: 可以！同一个手机号可以注册所有平台。

### Q: API 密钥会过期吗？
A: 不会过期，除非你主动删除或重置。

### Q: 免费额度多久刷新？
A: 免费额度是一次性的，用完即止。部分平台会定期赠送新额度。

### Q: 可以删除账号重新注册吗？
A: 不建议。平台会检测，可能导致无法再次获得免费额度。

## 安全提醒

### ⚠️ 重要！

1. **不要分享密码**
   - 不在聊天中分享（微信、QQ 等）
   - 不在邮件中发送
   - 不告诉他人

2. **使用密码管理器**
   - 为每个平台生成独立密码
   - 安全存储 API 密钥

3. **开启双重验证**
   - 短信验证
   - 谷歌验证器
   - 邮箱验证

4. **定期检查**
   - 查看账号登录记录
   - 检查 API 使用情况
   - 发现异常立即修改密码
