# 🔐 密码管理器使用指南

## 为什么需要密码管理器？

在注册多个 API 平台时，**强烈建议使用密码管理器**，原因如下：

### 安全风险

❌ **重复使用密码**：一个平台泄露，所有账号遭殃  
❌ **弱密码**：容易被暴力破解  
❌ **密码写在纸上/记事本**：容易丢失或被他人看到  
❌ **在聊天中分享密码**：如微信、QQ、邮件等  

### 密码管理器的好处

✅ **为每个平台生成独立强密码**  
✅ **自动填充密码**，无需记忆  
✅ **加密存储**，只有你能访问  
✅ **跨设备同步**，手机电脑都能用  
✅ **密码泄露提醒**，及时更换

## 使用教程（以 Bitwarden 为例）

### 第一步：注册账号

1. 访问 https://bitwarden.com
2. 点击「Create Account」
3. 输入邮箱和**主密码**（这个密码一定要记住！）
4. 验证邮箱

**主密码建议**:
- 至少 12 位
- 使用短语组合，如：`BlueCoffee$Mountain2024!`
- **这个密码必须记住**，无法找回！

---

### 第二步：安装浏览器扩展

1. 打开 Chrome 网上应用店
2. 搜索「Bitwarden」
3. 点击「添加至 Chrome」
4. 登录你的 Bitwarden 账号

---

### 第三步：保存第一个密码

1. 访问任意网站（如 https://console.volcengine.com/）
2. 注册账号并设置密码
3. Bitwarden 会提示「是否保存密码？」
4. 点击「保存」

或者手动添加：
1. 点击浏览器右上角 Bitwarden 图标
2. 点击「添加项目」
3. 选择「登录」
4. 填写：
   - 名称：火山引擎
   - 用户名：15802968897
   - 密码：点击「生成」创建强密码
5. 点击「保存」

---

### 第四步：自动填充密码

1. 再次访问火山引擎官网
2. 点击登录框
3. Bitwarden 自动提示保存的账号
4. 点击账号，自动填充密码
5. 登录成功！

## 生成强密码

### 好密码的特征

✅ 至少 16 位  
✅ 包含大小写字母  
✅ 包含数字  
✅ 包含特殊字符（!@#$%^&*）  
✅ 无规律，不可预测  

### 示例

```
✅ X7#mK9$pL2@nQ5!wR8
✅ BlueCoffee$Mountain2024!River
✅ 7xK#mP9$vL2@nQ5!
❌ 123456（太简单）
❌ password（常见密码）
❌ 15802968897（手机号）
❌ ai2046netCN（有规律）
```

## 应急计划

### 如果忘记密码管理器主密码

**Bitwarden**: 
- 可以设置紧急联系人
- 导出账号恢复码，安全保存

**1Password**: 
- 无法恢复，必须记住主密码
- 提前打印应急工具包

**建议**:
1. 将主密码告诉信任的家人
2. 写在纸上，存放在保险箱
3. 使用密码管理器的紧急访问功能
