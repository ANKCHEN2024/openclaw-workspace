## 🔑 快速配置

### 方式一：Web 界面
1. 访问 http://localhost:8080
2. 进入「设置」页面
3. 选择平台，输入 API 密钥
4. 保存配置

### 方式二：CLI
```bash
short-video config
# 按提示选择平台并输入密钥
```

### 方式三：API
```bash
curl -X POST http://localhost:3000/api/config \
  -H "Content-Type: application/json" \
  -d '{
    "volcengine_api_key": "your_key",
    "minimax_api_key": "your_key"
  }'
```

## 💡 多平台策略

建议配置 2-3 个平台，轮流使用：

```bash
# 配置火山引擎（主要）
short-video config volcengine

# 配置 MiniMax（备用）
short-video config minimax

# 配置快手（批量时用）
short-video config kuaishou

# 生成时指定平台
short-video generate "提示词" --provider volcengine
```

**好处**:
- 免费额度最大化
- 避免单平台限流
- 对比生成质量
- 降低使用成本

### Q: 能同时使用多个平台吗？
