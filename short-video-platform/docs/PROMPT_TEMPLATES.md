## 💡 提示词公式

### 基础公式
```
[主体] + [动作/状态] + [场景/背景] + [风格/质量]
```

### 进阶公式
```
[主体详细描述] + [具体动作] + [场景环境] + [镜头角度] + [光影效果] + [艺术风格] + [画质要求]
```

### 示例拆解
```
一只金色毛发的柯基犬 + 在绿茵场上欢快奔跑 + 蓝天白云背景 + 低角度跟拍 + 阳光明媚 + 写实风格 + 4K 高清
```

## ⚠️ 避免使用的词汇

❌ 过于抽象：「美好的」「伟大的」
❌ 矛盾描述：「静止的运动」「黑暗的光明」
❌ 复杂场景：超过 3 个主体
❌ 文字内容：AI 难以生成准确文字

## 📝 实战练习

### 练习 1：描述你的宠物
```
主体：你的宠物
动作：它最喜欢做的事
场景：它常去的地方
风格：你想要的效果
```

### 练习 2：描述旅行记忆
```
主体：印象最深的景点
动作：当时的场景
场景：周围环境
风格：照片/视频的风格
```

### 练习 3：产品创意
```
主体：你的产品
动作：展示产品特点
场景：适合的背景
风格：商业/艺术风格
```
