📝 通用注册步骤

1
点击「立即注册」
点击上方任意平台的注册按钮，会在新窗口打开官网。建议使用 Chrome 或 Edge 浏览器。
2
手机号注册
输入手机号，获取短信验证码。大部分平台支持中国大陆手机号（+86）。
3
设置密码
使用密码管理器生成强密码！建议：16 位以上，包含大小写字母、数字、特殊字符。

💡 示例密码格式：X7#mK9$pL2@nQ5!wR
4
实名认证
根据提示完成实名认证（需要身份证）。这是国家法规要求，所有正规平台都需要。
5
获取 API 密钥
登录控制台 → 找到「API 密钥管理」或「访问令牌」→ 创建新密钥 → 复制保存。

⚠️ 重要：API 密钥只显示一次，请立即复制保存！
6
配置到平台
回到我们的短视频生成平台 → 设置页面 → 选择对应平台 → 粘贴 API 密钥 → 保存。
