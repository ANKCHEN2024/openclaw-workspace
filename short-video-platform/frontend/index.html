📖 使用说明

  1. 填写视频描述：详细描述你想要生成的视频内容，包括场景、角色、动作等
  2. 上传参考图片（可选）：如果有参考图片，可以上传以获得更准确的效果
  3. 选择视频风格：从 8 种风格中选择最适合你需求的风格
  4. 设置视频时长：选择 5 秒到 60 秒不等的视频时长
  5. 点击生成：提交后系统将开始生成视频，你可以实时查看进度
  6. 查看结果：生成完成后会自动跳转到视频详情页，可以预览、下载或分享
💡 提示：
  • 描述越详细，生成效果越好
  • 首次使用请先在设置页面配置 API 密钥
  • 视频生成通常需要 1-3 分钟，请耐心等待
