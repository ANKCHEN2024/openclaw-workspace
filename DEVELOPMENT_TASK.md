## 技术栈（纯国产）

### AI 模型 API
| 模块 | 推荐 API | 备选 API |
|------|----------|----------|
| 大语言模型 | 阿里通义千问 | 百度文心一言、MiniMax |
| 图像生成 | 阿里通义万相 | 火山引擎即梦 AI |
| 视频生成 | 快手可灵 AI | 火山引擎即梦 AI、腾讯智影 |
| 语音合成 | 阿里云智能语音 | 火山引擎语音合成 |
| 语音识别 | 阿里云语音识别 | 腾讯云语音识别 |
| 音乐生成 | 网易云音乐 | QQ 音乐 |

### 后端技术
- Node.js + Express
- Redis（任务队列）
- PostgreSQL（数据库）
- MinIO（对象存储）

### 前端技术
- Vue 3 + Element Plus
- 响应式设计

## 开发团队组织

### Subagent 分工

1. **Subagent-1: 架构组**
   - 任务：项目结构设计、数据库设计
   - 交付：项目脚手架、数据库 Schema

2. **Subagent-2: 故事分析模块**
   - 任务：小说分析、自动分集、剧情摘要
   - API：阿里通义千问
   - 交付：story-analyzer 模块

3. **Subagent-3: 人物构建模块**
   - 任务：人物提取、人物设定、LoRA 训练
   - API：阿里通义万相
   - 交付：character-builder 模块

4. **Subagent-4: 场景构建模块**
   - 任务：场景分析、场景生成、一致性维护
   - API：火山引擎即梦 AI
   - 交付：scene-builder 模块

5. **Subagent-5: 分镜生成模块**
   - 任务：自动分镜、镜头设计、分镜脚本
   - API：阿里通义千问
   - 交付：storyboard-generator 模块

6. **Subagent-6: 视频生成模块**
   - 任务：Text-to-Video、镜头生成、运动控制
   - API：快手可灵 AI、火山引擎即梦 AI
   - 交付：video-generator 模块

7. **Subagent-7: 音频处理模块**
   - 任务：配音合成、语音识别、字幕生成
   - API：阿里云智能语音
   - 交付：audio-processor 模块

8. **Subagent-8: BGM 音效模块**
   - 任务：背景音乐生成、音效匹配
   - API：网易云音乐
   - 交付：bgm-generator 模块

9. **Subagent-9: 视频合成模块**
   - 任务：多轨道合成、转场、调色、渲染
   - API：FFmpeg
   - 交付：video-composer 模块

10. **Subagent-10: 前端开发组**
    - 任务：Web 界面、项目管理、视频预览
    - 技术：Vue 3
    - 交付：frontend 完整界面

11. **Subagent-11: 测试组**
    - 任务：单元测试、集成测试、E2E 测试
    - 交付：测试报告、Bug 列表

12. **Subagent-12: 文档组**
    - 任务：API 文档、使用手册、部署指南
    - 交付：完整文档
