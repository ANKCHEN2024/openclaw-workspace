## 📥 入场条件检查清单

### 多头入场 ✅
- [ ] H1 趋势向上（价格 > EMA50 > EMA200）
- [ ] 价格回调至 EMA20/50 或 Fib 50%/61.8%
- [ ] M15 出现看涨 Pin Bar 或 bullish engulfing
- [ ] 信号 K 线已收盘确认
- [ ] 止损距离合理（≤账户 2% 风险）
- [ ] 盈亏比 ≥ 1:3

### 空头入场 ✅
- [ ] H1 趋势向下（价格 < EMA50 < EMA200）
- [ ] 价格反弹至 EMA20/50 或 Fib 50%/61.8%
- [ ] M15 出现看跌 Pin Bar 或 bearish engulfing
- [ ] 信号 K 线已收盘确认
- [ ] 止损距离合理（≤账户 2% 风险）
- [ ] 盈亏比 ≥ 1:3

## 🛡️ 止损止盈设置

### 止损（二选一，取较近者）
```
□ 结构止损：摆动低点/高点外 $3-5
□ ATR 止损：1.5 × H1 ATR(14)
□ 最大止损：不超过账户 2% 风险
```

### 止盈（分批策略）
```
第一目标：50% 仓位 @ 1:1.5 盈亏比
第二目标：50% 仓位 @ 1:3 盈亏比 或 移动止盈

达到 1:1 后：止损移至保本价
```

### 移动止盈方法（第二目标）
```
□ 摆动点追踪：新高后移至前低
□ 均线追踪：EMA20 下方（多单）
□ ATR 追踪：最高价 - 2×ATR（多单）
```

## 💰 仓位计算

```
仓位（盎司）= (账户资金 × 1%) ÷ 止损距离（美元）

示例：
账户 $10,000 | 止损 $10
仓位 = ($10,000 × 0.01) ÷ $10 = 10 盎司 = 0.1 标准手
```

## 🚨 风控红线

### 单笔风险
```
□ 标准：1%
□ 最大：2%（绝不超越）
```

### 日亏损上限
```
□ 达到 3% → 停止当日交易
□ 连续 3 日触及 → 暂停 1 周
```

### 连续亏损处理
```
□ 连续 3 亏 → 半仓 + 暂停 4 小时
□ 连续 5 亏 → 1/4 仓 + 暂停 24 小时
□ 连续 7 亏 → 停止 1 周 + 全面复盘
```

### 其他限制
```
□ 最大日交易次数：3-5 笔
□ 单一品种仓位：≤50% 保证金
□ 实际杠杆：≤10:1
```

## 📱 快速计算工具

### 盈亏比计算
```
盈亏比 = (止盈价 - 入场价) ÷ (入场价 - 止损价)

示例：
入场 $2650 | 止损 $2640 | 止盈 $2680
盈亏比 = ($2680-$2650) ÷ ($2650-$2640) = 30÷10 = 1:3 ✓
```

### 风险金额计算
```
风险金额 = 仓位（盎司）× 止损距离（美元）

示例：
10 盎司 × $10 = $100 风险
```

### 仓位反推
```
仓位 = 风险金额 ÷ 止损距离

示例：
想风险$100 | 止损$15
仓位 = $100 ÷ $15 = 6.6 盎司
```
