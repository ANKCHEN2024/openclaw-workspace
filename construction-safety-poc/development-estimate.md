# 开发时间估算

**项目名称**：建筑工地安全视频分析系统

### Phase 1: 真实数据验证
**时间**：2 周  
**人员**：2 人（后端开发 + 算法工程师）  
**目标**：验证算法在真实场景的准确性

| 任务 | 工时 (人天) | 负责人 | 依赖 |
|------|-------------|--------|------|
| 收集真实 CAD 图纸 (3-5 个项目) | 3 | 后端 | 客户配合 |
| 测试 ezdxf/OpenDesign 解析 | 2 | 后端 | - |
| 验证 CAD 解析准确性 | 2 | 算法 | CAD 图纸 |
| 优化摄像头规划算法 | 3 | 算法 | CAD 解析 |
| 生成点位规划报告 | 2 | 后端 | 算法优化 |
| 客户现场勘测验证 | 2 | 后端 + 算法 | 客户安排 |
| **小计** | **14 人天** | **2 周** | |

**里程碑**：
- [ ] CAD 解析准确率 ≥ 90%
- [ ] 点位规划覆盖率 ≥ 85%
- [ ] 客户确认技术方案

**风险**：
- ⚠️ CAD 图纸格式不统一（需适配多种版本）
- ⚠️ 客户配合度（现场勘测需协调）

---

### Phase 2: AI 模型训练
**时间**：4 周  
**人员**：3 人（算法工程师 x2 + 数据标注员）  
**目标**：训练高精度 AI 检测模型

| 任务 | 工时 (人天) | 负责人 | 依赖 |
|------|-------------|--------|------|
| 收集工地安全数据集 | 5 | 算法 + 标注 | 网络爬虫/购买 |
| 数据清洗与标注 | 10 | 标注员 | 数据集 |
| 训练安全帽检测模型 | 3 | 算法 | 标注数据 |
| 训练反光衣检测模型 | 3 | 算法 | 标注数据 |
| 训练烟火识别模型 | 3 | 算法 | 标注数据 |
| 训练入侵检测模型 | 2 | 算法 | 标注数据 |
| 模型优化与调参 | 5 | 算法 | 初步模型 |
| 模型评估与测试 | 3 | 算法 | 优化模型 |
| 模型部署测试 | 2 | 算法 + 后端 | 测试环境 |
| **小计** | **36 人天** | **4 周** | |

**里程碑**：
- [ ] 数据集规模 ≥ 10,000 张图片
- [ ] 安全帽检测准确率 ≥ 95%
- [ ] 反光衣检测准确率 ≥ 93%
- [ ] 烟火识别准确率 ≥ 92%
- [ ] 单模型推理延迟 ≤ 100ms

**风险**：
- ⚠️ 数据质量不足（需购买商业数据集）
- ⚠️ 模型过拟合（需数据增强）
- ⚠️ 边缘场景识别率低（需针对性优化）

**预算**：
- 数据集购买：¥20,000 - ¥50,000
- 标注外包：¥15,000 - ¥30,000
- GPU 云服务：¥10,000 - ¥20,000
- **小计**：¥45,000 - ¥100,000
