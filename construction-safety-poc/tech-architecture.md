### 2. 摄像头点位规划算法

```
┌──────────────────────────────────────────────────────────────┐
│                    摄像头规划算法流程                          │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  输入：CAD 解析结果 (JSON)                                    │
│       ↓                                                      │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  Step 1: 初始化建筑平面网格 (1m x 1m)                   │   │
│  └──────────────────────────────────────────────────────┘   │
│       ↓                                                      │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  Step 2: 放置关键点位摄像头 (出入口/通道/角落)           │   │
│  └──────────────────────────────────────────────────────┘   │
│       ↓                                                      │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  Step 3: 射线追踪计算可见区域                           │   │
│  │  ┌────────────────────────────────────────────────┐   │   │
│  │  │  for each pixel:                               │   │   │
│  │  │    cast ray from camera position               │   │   │
│  │  │    check intersection with walls               │   │   │
│  │  │    mark visible/invisible                      │   │   │
│  │  └────────────────────────────────────────────────┘   │   │
│  └──────────────────────────────────────────────────────┘   │
│       ↓                                                      │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  Step 4: 计算盲区 & 覆盖率                              │   │
│  └──────────────────────────────────────────────────────┘   │
│       ↓                                                      │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  Step 5: 遗传算法优化点位                              │   │
│  │  ┌────────────────────────────────────────────────┐   │   │
│  │  │  population = random_camera_positions()        │   │   │
│  │  │  for generation in range(100):                 │   │   │
│  │  │    fitness = calculate_coverage(population)    │   │   │
│  │  │    select_best()                               │   │   │
│  │  │    crossover() + mutate()                      │   │   │
│  │  └────────────────────────────────────────────────┘   │   │
│  └──────────────────────────────────────────────────────┘   │
│       ↓                                                      │
│  输出：摄像头位置清单 + 覆盖率报告                             │
│                                                              │
└──────────────────────────────────────────────────────────────┘

算法复杂度:
├── 射线追踪：O(n × m), n=摄像头数，m=网格点数
├── 遗传算法：O(g × p × n), g=代数，p=种群大小，n=摄像头数
└── 优化目标：覆盖率≥95%, 摄像头数量最小化
```

## 技术选型总结

| 层级 | 组件 | 技术选型 | 备选方案 |
|------|------|----------|----------|
| **前端** | 监控大屏 | Vue3 + ECharts | React + AntV |
| | 移动端 | UniApp | Flutter |
| | 管理后台 | Vue3 + Element Plus | React + Ant Design |
| **后端** | API 框架 | FastAPI (Python) | Spring Boot (Java) |
| | 网关 | Nginx + Kong | APISIX |
| | 消息队列 | Redis Streams | RabbitMQ / Kafka |
| **AI** | 目标检测 | YOLOv8 | Detectron2 |
| | 推理引擎 | NVIDIA Triton | TensorRT |
| | 模型训练 | PyTorch | TensorFlow |
| **数据** | 关系数据库 | PostgreSQL | MySQL |
| | 缓存 | Redis | Memcached |
| | 时序数据 | TDengine | InfluxDB |
| | 文件存储 | MinIO | Ceph / NAS |
| **CAD** | DWG 解析 | OpenDesign Teigha | ezdxf (DXF only) |
| | 几何计算 | Shapely + NumPy | CGAL |
| **部署** | 容器化 | Docker + K8s | Docker Swarm |
| | CI/CD | GitLab CI | Jenkins |
| | 监控 | Prometheus + Grafana | Zabbix |
